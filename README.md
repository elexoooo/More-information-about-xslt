Guidelines and Social Web Technology
Why we use Social Media and Web Tech?
Using social media and web technology is important because it makes talking and sharing things very easy for everyone. It helps people stay connected and see all updates or announcements in one place. The goal is to make communication fast and help people work together as a group.
About the Design and Media
The design is simple because we want it to be easy for everyone to use, even beginners. We focus on how to use the media and the tools to communicate, more than just how the page looks. A simple layout helps people find what they need without getting lost or confused.
Main Features:
• Easy Communication: It’s very simple to send messages and talk.
• Simple Layout: You can find all the main features easily.
• Accessibility: It works on phones and computers so everyone can join.
Evaluation:
• Advantages: Good interaction, works for groups, and it is secure.
• Disadvantages: You can’t change the profile design much, and it might be a bit hard at first for some people.
Recommendations:
• Make a clear structure for the channels.
• Share clear rules for how to communicate.
• Use simple guides to help people start
